# Lotus for iOS

Welcome to Lotus's iOS repository. Lotus is a mobile application for iOS 12 and above to allow users to track what they are trying to be clean from.

## Intended Features
- Ability to track what you want to be sober from (Not Yet Implemented) 
- Ability to reset the counter and view your highest streak (Not Yet Implemented) 
- Ability to import/export from/to the Lotus for Android (Not Yet Implemented)
- Will be able to write notes about each set back and what went wrong with links to support (Not Yet Implemented)
- Medals/Tokens to show your progession is a way that motivates you (Not Yet Implemented)
